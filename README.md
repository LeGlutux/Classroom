Learning purpose repository

# Requirements

- Terminal / Console
- NPM > 6
- Chrome
- Git
- Visual Studio Code

Recommended plugins VSC:
  - Prettier - Code formatter
  - Tailwind CSS IntelliSense
  - TSLint
  - Live Share (Microsoft)

# Installation 

```
git clone git@github.com:poulainv/tuto-leo.git
cd tuto-leo
npm install
npm start
```

Open browser http://localhost:3000
